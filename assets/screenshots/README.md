## How to Add Screenshots

1. Run the app: `pnpm run dev`
2. Navigate through each screen
3. Take screenshots using:
   - macOS: `Cmd + Shift + 4` or `screencapture -i filename.png`
   - Windows: `Win + Shift + S` or Snipping Tool
   - Linux: `Print Screen` or Flameshot

4. Replace the placeholder images with actual screenshots
