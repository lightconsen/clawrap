# OpenClaw Clawrap 优化 Todo

基于用户痛点调研报告的 Settings 可优化点

## 一、安全风险类 (高优先级)

### 1. 权限管控中心 🔐
**痛点**: 58% 用户因数据泄露担忧放弃使用，OpenClaw 默认获取高级系统权限

**Settings 优化方案**:
- [x] **权限仪表盘** - Overview 页面增加权限使用概览
  - 文件系统访问记录
  - 网络请求统计
  - 环境变量读取日志
- [x] **权限开关** - 新增 "Permissions" 设置页
  - 文件系统访问：只读/指定目录/完全访问
  - 网络请求：白名单模式
  - 系统命令执行：确认提示开关
- [ ] **技能安全审计** - Skills 页面增强
  - 标记已验证安全技能
  - 显示技能权限请求
  - 恶意技能黑名单

### 2. 数据隐私保护 🛡️
**痛点**: 41.44% 用户听说"会自动删文件"不敢尝试

**Settings 优化方案**:
- [ ] **操作审计日志** - 新增 "Audit Log" 页面
  - 所有文件操作记录
  - 可追溯、可撤销
  - 支持导出审计报告
- [ ] **敏感操作确认** - 全局设置
  - 删除文件前二次确认
  - 敏感 API 调用授权
  - 操作通知开关

## 二、成本控制类 (高优先级)

### 3. Token 消耗监控 💰
**痛点**: Token 消耗像"无底洞"，学生用户"几分钟内花费十多块"

**Settings 优化方案**:
- [x] **用量仪表盘** - Overview 页面增加
  - 今日/本周/本月 Token 消耗
  - 实时费用估算 (USD/CNY)
  - 消耗趋势图表
- [ ] **预算告警** - 新增 "Usage" 设置页
  - 日/周/月预算上限设置
  - 达到 80% 预算时告警
  - 超预算自动暂停
- [ ] **模型成本对比** - Models 页面增强
  - 显示各模型每百万 Token 价格
  - 历史消耗统计 per 模型
  - 推荐性价比模型

### 4. 智能模型路由 🤖
**痛点**: "便宜的模型效果一般，效果不错的模型太贵"

**Settings 优化方案**:
- [ ] **任务 - 模型匹配规则**
  - 简单任务 → 便宜模型
  - 复杂任务 → 高端模型
  - 图片识别 → 视觉模型
- [ ] **模型降级策略**
  - 高端模型失败时自动降级
  - 降级阈值配置
